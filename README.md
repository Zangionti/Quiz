# Квиз по столицам
Интерактивная викторина на знание столиц стран мира. Игрок вводит своё имя, отвечает на 10 вопросов с 4 вариантами ответа и получает результат.

## Правила игры
- Введи своё имя и нажми "Начать"
- Отвечай на 10 вопросов про столицы стран
- За каждый правильный ответ начисляется 1 балл
- В конце показывается итоговый счёт

## Функциональные возможности
Проверка ввода имени перед началом игры
Отображение вопроса и 4 вариантов ответа
Подсчёт правильных ответов
Сохранение результата в файл results.txt

## Руководство администратора
Windows 10/11
.NET Framework 4.7.2
Visual Studio 2022
Открыть файл Quiz.sln и запустить проект

## Руководство пользователя
1. Запустить программу
2. Ввести имя в поле и нажать "Начать"
3. Нажимать на кнопку с правильным ответом (желательно)
4. После 10 вопросов появится результат

## Структура классов
- `Question` — хранит текст вопроса, варианты ответа и индекс правильного ответа
- `Player` — хранит имя игрока и счёт
- `MainForm` — главное меню
- `QuizForm` — экран викторины
